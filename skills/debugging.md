# Debugging Skill

Purpose:
Find and fix bugs automatically.

Debugging Workflow:

1 reproduce bug
2 check logs
3 isolate failing component
4 inspect stack trace
5 fix code

Tools:

logs
stack traces
unit tests
integration tests

Best Practices:

• fix root cause
• add tests after fix
• avoid temporary patches
