# Frontend Development Skill

Purpose:
Generate scalable and maintainable frontend UI.

Technologies:

HTML
CSS
JavaScript
React
Tailwind

Best Practices:

• Use component-based architecture
• Separate UI and logic
• Use reusable UI components
• Optimize images and assets
• Use lazy loading

UI Workflow:

1 analyze UI prototype
2 split UI into components
3 generate responsive layout
4 connect UI to API
5 optimize performance

Page Types:

index page
pricing page
dashboard
auth pages
proxy marketplace
